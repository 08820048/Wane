# Contributing

Wane is intentionally small: pure AppKit/SwiftUI, no third-party dependencies, and no network requests.

Before opening a pull request:

- Keep changes scoped to the stated issue or feature.
- Run `swift test` when touching core logic.
- Run `swift build` before submitting UI or app lifecycle changes.
